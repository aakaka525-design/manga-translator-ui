# v2.0.8 更新日志

发布日期：2026-01-19

## ✨ 新功能

- **Qt 编辑器多选批量样式设置**：支持选中多个文本框后批量修改字体、颜色、大小、对齐等样式属性
- **增强的字体颜色选择器**：
  - 显示当前颜色的 RGB 值
  - 支持复制/粘贴颜色功能，可快速将一个文本的颜色应用到其他文本
  - 新增常用颜色保存功能，最多保存 20 个常用颜色
  - 常用颜色持久化保存，关闭软件后不会丢失
  - 颜色选择对话框自动加载保存的常用颜色
- **自定义 API 参数功能**：
  - 支持为 OpenAI、Gemini 等翻译器传递自定义 API 参数
  - 可通过 UI 界面勾选开关并点击"打开文件"按钮编辑配置
  - 配置文件位于 `examples/custom_api_params.json`
  - 支持实时生效，修改配置后下次翻译立即使用新参数
  - 适用于控制 Ollama 等本地模型的特殊参数（如关闭思考模式）

## 🐛 修复

- **Qt 编辑器选择和交互**：修复框选残留、Ctrl+多选显示、光标样式等问题，简化选择逻辑提高稳定性
- **Qt 编辑器右键菜单**：修复多选时右键打开菜单会取消选择的问题
- **Qt 编辑器保存 JSON**：修复保存 JSON 时"正在保存"提示不消失的问题
- **Qt 编辑器复制粘贴**：修复新增/粘贴文本框时重复添加两个的问题
- 修复高质量翻译会覆盖替换翻译流程的 bug
- 修复 OpenAI 翻译器被 Cloudflare 拦截的问题
- 修复 `packaging/launch.py` 中 `continue` 语法错误
- 修复 ONNX 修复模型缺少日志输出的问题
- 修复日志输出缓冲导致的卡住问题
- 修复 `ignore_bubble` 功能的一致性问题

## 🔧 优化

- 优化 Qt 编辑器日志级别，减少噪音
- 优化安装脚本的删除提醒
- 优化并发流水线翻译性能
- 优化 OpenAI 翻译器重试机制
